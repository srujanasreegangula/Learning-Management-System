# Learning Management System (Frontend)

## Overview

The Learning Management System (LMS) is a frontend-only web application developed using HTML, CSS, and JavaScript. It provides a modern and responsive interface where students can explore courses, access lessons, monitor their learning progress, and manage assignments. This project focuses on creating an intuitive user experience without using any backend technologies.

## Features

* Student Dashboard
* Login & Registration UI
* Course Catalog
* Course Details Page
* Lesson Navigation
* Progress Tracking
* Assignment Section
* Quiz Interface
* Student Profile
* Fully Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
